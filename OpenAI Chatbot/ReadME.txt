# OpenAI Chatbot Widget

A clean, lightweight **OpenAI Chatbot Widget** you can embed on any webpage to interact with ChatGPT models for learning, experimentation, and portfolio demonstration.

![screenshot](https://via.placeholder.com/600x300?text=Chatbot+Demo)

---

## 🚀 Features

✅ Uses **OpenAI Chat Completion API** (`gpt-4o-preview` by default)  
✅ Clean, mobile-friendly floating widget UI  
✅ User and assistant message styling  
✅ Handles loading states and error messaging gracefully  
✅ Modern **async/await + fetch** implementation  
✅ Great for building API and frontend integration skills

---

## 🛠️ Tech Stack

- **HTML / CSS / JavaScript** (Vanilla, no frameworks)
- **OpenAI Chat API**
